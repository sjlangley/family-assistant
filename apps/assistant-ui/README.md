# Assistant UI

Minimal LLM UI scaffold using Vite, React, TypeScript, Tailwind, Vitest, and Prettier.

## Prerequisites

- Node.js 22+
- npm 10+

## Install

```bash
npm install
```

## Run locally

```bash
npm run dev
```

## Run tests

```bash
npm run test
```

## Format

```bash
npm run format
```

## Run with Docker

```bash
docker build -t assistant-ui .
docker run --rm -p 3000:3000 assistant-ui
```
